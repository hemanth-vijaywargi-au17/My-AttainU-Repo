Max Sum Problem

Using Kadane's Algorithm

Time Complexity : O(n) , Space Complexity : O(1) 

    
    def max_sum(arr):
        max_sum = arr[0]
        cur_sum = arr[0]
        for i in range(1, len(arr)):
            cur_sum = max(cur_sum + arr[i], arr[i])
            max_sum = max(max_sum, cur_sum)
        return max_sum
